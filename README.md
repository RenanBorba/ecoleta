<br><br>

Aplicação Full MVC desenvolvida para o site responsivo Ecoleta que é voltado para cadastro e busca de pontos de coleta de materiais recicláveis e resíduos. ♻️

<br><br>

## :rocket: Tecnologias
### Front-end
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>Google Fonts</li>
  <li>Javascript</li>
  <li>IBGE API de Localidades</li>
  <li>Fetch API</li>
</ul>

<br>

### Back-end
<ul>
  <li>Node.js</li>
  <li>Nodemon</li>
  <li>Routes</li>
  <li>Express</li>
  <li>Template Engine</li>
  <li>Nunjucks</li>
  <li>Sqlite</li>
</ul>

<br><br><br>

## :arrow_forward: Start
<ul>
  <li>npm install</li>
  <li>npm run start / npm start</li>
</ul>
